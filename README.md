📊 **Zomato Restaurants Analysis Dashboard | Data Analytics Project**

Excited to share my latest **Power BI Dashboard** on **Zomato Restaurants Analysis**, built using the **Zomato Restaurants in India** dataset.

This project focuses on analyzing restaurant trends across major cities and provides valuable insights into customer preferences, cuisines, ratings, and online services.

### 🔍 Key Insights:

✅ Analyzed **1,102 restaurants** across **82 cities** and **14 countries**.
✅ Identified the **highest-rated cuisines** and the most popular restaurant categories.
✅ Compared **online delivery** and **table booking** availability.
✅ Visualized **city-wise restaurant distribution** using maps.
✅ Explored restaurant ratings, customer votes, and cuisine popularity.
✅ Built an interactive dashboard with filters for easy data exploration.

### 🛠️ Tools & Technologies:

* Power BI
* Microsoft Excel / CSV
* Data Cleaning
* Data Visualization
* Exploratory Data Analysis (EDA)

This project enhanced my skills in transforming raw data into meaningful business insights through interactive visualizations.
